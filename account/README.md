## 项目介绍

广学账号是广学五题坊的账号系统（和未来的综合API服务系统），用户可以在账号系统中注册账号、登录账号、找回密码等。

## 项目架构

- 前端：HTML + Bootstrap
- 后端：Node.js + Express + MySQL
- 数据库：MySQL

注意这是我们所有新项目中唯一一个保留了旧技术栈Express的项目（因为这个项目重视后端而非前端），后续所有项目都会使用Next.js。同时也请大家注意Bootstrap和Tailwind CSS的用法存在差异。
