# 你说的对，但是

党的二十大报告指出，要“全面提高人才自主培养质量，着力造就 **拔尖创新人才**”。北京师范大学附属实验中学校长李晓辉在新学年致辞提到：**荣誉，是我们共同的骄傲；创新，是我们永恒的追求**。在“荣誉·创新——我与实验共奋进”这一学年主题下，北师大实验中学高一 14 班为弘扬竞赛特色，发扬自主创新精神，在本班公众号上开设周期性栏目 **“广学五题坊”**。

每一周，五个学科竞赛每个学科会发布一道题在公众号上。大家可以提交解答，解答形式不限（文字、图片、视频、文件等皆可），每周会公布上周题目的优秀解答。奖励机制尚不成熟；题目不保证原创性，所有题目不可作为商业用途。题目定位于中学生和学科竞赛爱好者。欢迎大家踊跃参与。

我们希望朋友们通过本栏目能够：

+ 徐徐渐进
+ 广见洽闻
+ 业精于勤

网站实现了查看栏目、题库、提交记录、排行榜，提交题目，体验小游戏等功能，可以通过该网站沉浸式学习和体验。详细功能介绍请查阅[《广学五题坊使用手册》](/main/manual.md)。

值得注意的是，我们十分重视语文老师 **Boo** 提出的“将语文融入广学五题坊”的宝贵意见。考虑到语文等人文学科对于竞赛和理科的帮助，我们积极贯彻落实相关建议，仔细研究学科之间的关系，抓住用户的兴趣，目前开发了两款游戏：分别是 **诗词九宫格** 和 **Hangman**。

- 诗词九宫格是一款以诗词大会中一类题型为灵感的游戏设计，考验玩家从宫格中唯一正确的诗词句，范围是高中必背古诗词曲，旨在锻炼玩家对于课本中诗词的熟悉程度。

- Hangman 是一款西方古老的民间游戏，要求玩家在 6 次尝试机会内猜出一个单词，范围是高考英语 3500 词，旨在锻炼玩家对单词的熟练度和逻辑推理能力。

游戏题库范围和游戏种类目前比较有限，之后可以继续发展完善。也欢迎有与学科游戏相关的建议的同学将你的宝贵意见提供给我们。

不难发现，网站的制作花费了不少经费和时间，我们的公众号也开通了打赏功能，所以欢迎各位用户在今后的栏目中支持我们 **（未满十八周岁打赏需要经过监护人同意）**，我们今后也会在栏目中插入一些广告，请大家见谅。
